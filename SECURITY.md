# Security policy

If you find a security issue in Hunk, please report it privately.

## Reporting

Open a private security advisory on GitHub or contact the maintainers through the repository security channel if one is available.

Please include:

- affected version or commit
- impact summary
- reproduction steps or proof of concept
- any suggested mitigation

## What to expect

- we will acknowledge receipt as soon as practical
- we will validate the report and assess impact
- we will coordinate a fix and disclosure timeline when needed

Please do not post undisclosed vulnerabilities in public issues.
